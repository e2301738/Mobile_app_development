Meeting Calendar Application

This application allows you to manage your meetings efficiently.

Features:
- Save meeting data (title, place, date, time) to a local SQLite database.
- Add participants to meetings, including their photos.
- Photos are stored as BLOB data in the SQLite database.
- View meeting summaries with participant images and names.
- Search for meetings by various criteria.
- Customize the user interface theme (colors, fonts, sizes).
- Export and import data via JSON files.

Database Information:
The application uses an SQLite database (meeting_calendar.db) stored in the app-specific directory:
/Android/data/fi.christian.meeting_calendar/files/data/app_db/

How to use:
1. Fill in meeting details on the main screen.
2. Click 'Add Participants' to select people and their photos.
3. Use the 'Submit' button to save the meeting to the database.
4. Access 'Summary' to see all scheduled meetings and participant photos.
5. Use 'Settings' to change the application's appearance.
